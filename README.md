# C Programming: A Modern Approach

This repository contains my progress, exercises, and projects as I work through the **C Programming** course, primarily following the curriculum laid out in *C Programming: A Modern Approach* by Madhunandana H M.

## 📚 Course Syllabus

### Phase 1: The Fundamentals
* **Chapter 1: Introducing C** – History, strengths, and weaknesses.
* **Chapter 2: C Fundamentals** – Writing your first program, directives, functions, and variables.
* **Chapter 3: Formatted Input/Output** – Mastering `printf` and `scanf`.
* **Chapter 4: Expressions** – Arithmetic, assignment, and increment/decrement operators.
* **Chapter 5: Selection Statements** – Logical expressions, `if` statements, and `switch` cases.
* **Chapter 6: Loops** – `while`, `do-while`, and `for` statements.

### Phase 2: Data Types and Basic Structures
* **Chapter 7: Basic Types** – Integer, floating-point, and character types.
* **Chapter 8: Arrays** – One-dimensional and multidimensional arrays.
* **Chapter 9: Functions** – Function definitions, arguments, and return values.
* **Chapter 10: Program Organization** – Local vs. Global variables and blocks.

### Phase 3: Advanced Concepts (Pointers & Memory)
* **Chapter 11: Pointers** – Pointer variables and the address/indirection operators.
* **Chapter 12: Pointers and Arrays** – Pointer arithmetic and processing arrays.
* **Chapter 13: Strings** – String literals, variables, and the C String Library.
* **Chapter 14: The Preprocessor** – Macros and conditional compilation.
* **Chapter 15: Writing Large Programs** – Source files, header files, and makefiles.

### Phase 4: Data Structures and System Features
* **Chapter 16: Structures, Unions, and Enumerations** – Creating custom data types.
* **Chapter 17: Dynamic Storage Allocation** – `malloc`, `calloc`, `realloc`, and linked lists.
* **Chapter 18: Declarations** – Storage classes and type qualifiers.
* **Chapter 22: Input/Output** – Streams, file operations, and error handling.
